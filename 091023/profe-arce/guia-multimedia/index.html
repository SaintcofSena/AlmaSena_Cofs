Fuentes CSS El CSS de color propiedad define el color del texto para ser utilizado para el elemento HTML. El CSS font-family propiedad define el tipo de letra que se utilizara para el elemento HTML. El CSS font-size propiedad define el tamaño del texto que se utilizará para el elemento HTML.

Contacto

El atributo alt El atributo alt especifica un texto alternativo para una imagen, si la imagen no se puede visualizar. El atributo alt proporciona información alternativa para una imagen, si un usuario por alguna razón no puede verla ( debido a la conexión lenta, un error en el atributo src, o si el usuario utiliza un lector de pantalla).
